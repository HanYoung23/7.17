  10. 주요 태그

    • 단락 - p
      • p 태그를 사용 시 각 단락들을 구분하고 줄바꿈한다.

    • 줄바꿈 - br
      • br 태그를 사용 시 단순히 줄바꿈한다.

    • 이미지 - img
      • img + src / height / width / alt (이미지 사용x일 때 메세지) / title 등으로 이미지를 삽입한다.

    • 표 - table
      • table 태그로 tl, tr, td 태그를 묶어서 사용한다. border 속성으로 테두리를 꾸민다.
